# Conversor de Moedas

Este é um aplicativo de conversão de moedas desenvolvido em Java para estudos de consumo de API.

## Funcionalidades

- Conversão de valores entre diferentes moedas.

## Tecnologias Utilizadas

- Java
- API ExchangeRate
